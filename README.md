# SwiftData Demo

Небольшой демонстрационный проект для знакомства с **SwiftData**. 

## Основные концепции
- `@Model` — модель данных
- `@Query` — автоматическое получение данных
- `modelContext` — вставка и удаление объектов
- автоматическое обновление UI

## Когда использовать
- SwiftUI-проекты
- новые приложения
- минимум iOS 17

## SwiftData vs Core Data
SwiftData — декларативная обёртка над Core Data с меньшим количеством кода

## Ключевые концепции SwiftData

### `@Model`
Декларативное описание persistent-модели.  
SwiftData автоматически создаёт схему хранения и следит за изменениями.

### `modelContext`
Контекст работы с данными:
- вставка (`insert`)
- удаление (`delete`)
- отслеживание изменений

### `@Query`
Реактивный механизм получения данных.
UI автоматически обновляется при изменениях в хранилище.

В репозитории используется простая схема ветвления:

- `main`  
  Базовая версия проекта.  
  Используется как стартовая точка и референс.

- `feature/add-swiftdata-demo`  
  Демонстрационная ветка с реализацией SwiftData и данным README.  
  Используется для ознакомления с принципами работы SwiftData и выполнения практического задания.
